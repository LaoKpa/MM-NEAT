The following command uses saved files to create a neural style transfer image from a CPPN image:
python .\neural_style.py --content ..\..\..\data\imagematch\cat.jpg --styles ..\..\..\data\imagematch\supercreepypersonimage.jpg --output TEMP.jpg --iterations 100 --style-layer-weight-exp 0.2

This next command takes multiple style files and one content image and sends them to a different Python program at the same time. The result is that one combo image is made for each style
python .\neural_style_json.py --content ..\..\..\data\imagematch\cat.jpg --styles ..\..\..\data\imagematch\supercreepypersonimage.jpg ..\..\..\data\imagematch\sadsheepimage.bmp
